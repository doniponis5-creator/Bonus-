## 🔧 ҚАДАММА-ҚАДАМ ЎРНАТИШ

### ҚАДАМ 1: Расширение яратиш

1. **1С:Предприятие** ни **Конфигуратор** режимида очинг
2. Меню: **Конфигурация → Расширения конфигурации**
3. **"+"** (Добавить) босинг
4. Тўлдиринг:
   - **Имя:** `SBonusPlus`
   - **Синоним:** `SBonus+ Интеграция`
   - **Назначение:** `Дополнение`
   - **Префикс:** `SB_`
5. **OK** босинг

---

### ҚАДАМ 2: Умумий модул қўшиш

1. Расширение дарахтида: ПКМ **"Общие модули"** → **Добавить**
2. Модул хусусиятларини тўлдиринг:
   - **Имя:** `SB_ОбщийМодуль`
   - **Синоним:** `SBonus Общий Модуль`
   - Флаглар:
     - ✅ **Сервер** — ёқиш
     - ✅ **Вызов сервера** — ёқиш
     - ❌ **Клиент** — ўчириш
     - ❌ **Внешнее соединение** — ўчириш
     - ✅ **Привилегированный** — ёқиш
3. Модулни икки марта босиб очинг
4. `SB_CommonModule.bsl` файлидаги **барча кодни** кўчириб ёпиштиринг

---

### ҚАДАМ 3: Константалар яратиш

Расширение дарахтида: ПКМ **"Константы"** → **Добавить** — 3 та константа яратинг:

#### Константа 1: SB_APIUrl
- **Имя:** `SB_APIUrl`
- **Тип:** Строка (узунлик 200)
- **Қиймат:** `https://api.smartcentr.store`

#### Константа 2: SB_WebhookSecret
- **Имя:** `SB_WebhookSecret`
- **Тип:** Строка (узунлик 200)
- **Қиймат:** *(махфий калит — ҚАДАМ 7 да кўринг)*

#### Константа 3: SB_IntegrationEnabled
- **Имя:** `SB_IntegrationEnabled`
- **Тип:** Булево
- **Қиймат:** `Истина`

---

### ҚАДАМ 4: Регламентли вазифалар яратиш

Расширение дарахтида: ПКМ **"Регламентные задания"** → **Добавить**

#### Вазифа 1: Товарлар синхронизацияси (ҳар куни 06:00)
- **Имя:** `SB_СинхронизацияТоваров`
- **Имя метода:** `SB_ОбщийМодуль.СинхронизацияТоваровВSBonus`
- **Расписание:** Ҳар куни, 06:00
- **Использование:** ✅

#### Вазифа 2: Қолдиқларни янгилаш (ҳар соат)
- **Имя:** `SB_ОбновлениеОстатков`
- **Имя метода:** `SB_ОбщийМодуль.ОбновлениеОстатковВSBonus`
- **Расписание:** Ҳар соат (00 дақиқа)
- **Использование:** ✅

---

### ҚАДАМ 5: Чек учун подписка яратиш

1. Расширение дарахтида: ПКМ **"Подписки на события"** → **Добавить**
2. Тўлдиринг:
   - **Имя:** `SB_ПриПроведенииЧека`
   - **Источник:** `ДокументОбъект.ЧекККМ`
   - **Событие:** `ОбработкаПроведения`
   - **Обработчик:** `SB_ОбщийМодуль.ПриПроведенииЧекаSBonus`

---

### ҚАДАМ 6: Сақлаш ва янгилаш

1. **Ctrl+S** (расширениени сақлаш)
2. Меню: **Конфигурация → Обновить конфигурацию базы данных** (F7)
3. Янгилашни тасдиқланг

---

### ҚАДАМ 7: VPS ни созлаш (SBonus+ бэкенд)

VPS га киринг ва `.env.production` га қўшинг:

```bash
ssh root@145.223.100.16

# Махфий калит генерация қилиш
openssl rand -hex 32

# Натижани эслаб қолинг — бу сизнинг WEBHOOK_1C_SECRET

# .env.production га қўшиш
cd /opt/sbonus
nano .env.production
```

Қаторларни қўшинг:
```
ENABLE_1C_WEBHOOK=true
WEBHOOK_1C_SECRET=<генерация_қилинган_калитни_қўйинг>
```

API ни қайта ишга туширинг:
```bash
docker compose -f docker-compose.prod.yml --env-file .env.production up -d --build api
```

**МУҲИМ:** VPS даги `WEBHOOK_1C_SECRET` қиймати 1С даги `SB_WebhookSecret` константаси билан **бир хил** бўлиши керак!

---

### ҚАДАМ 8: Уланишни текшириш

1. **1С:Предприятие** ни (фойдаланувчи режими) очинг
2. Консоль кодини очинг
3. Бажаринг:
```
SB_ОбщийМодуль.ТестСоединения();
```
4. Чиқиши керак: `✅ Соединение с SBonus+ API успешно!`

5. Товар юбориш тести:
```
SB_ОбщийМодуль.ТестОтправкиТовара();
```
6. Чиқиши керак: `✅ Тестовый товар успешно отправлен!`

## ⚠️ МУҲИМ ЭСЛАТМАЛАР

### Нарх турлари
Кодда ишлатилган нарх турлари:
- `"Розничная"` — сотиш нархи
- `"Закупочная"` — таннарх

**Агар сизда бошқача номлар бўлса**, модул кодини очинг ва ўзгартиринг:
```
Справочники.ВидыЦен.НайтиПоНаименованию("Розничная")
Справочники.ВидыЦен.НайтиПоНаименованию("Закупочная")
```

Масалан, агар сизда "Оптовая" бўлса — `"Розничная"` ўрнига `"Оптовая"` ёзинг.

### Бир нечта филиал
Агар SBonus+ да бир нечта филиал бўлса, `ОтправитьДанныеЧека` функциясидаги қаторни uncomment қилинг:
```
// Данные.Вставить("branch_id", "ваш-branch-uuid");
```

### Логлар
Барча ҳаракатлар 1С Журнал регистрациясига ёзилади:
- `SBonus.Синхронизация` — товар синхронизацияси
- `SBonus.Чек` — чек юбориш
- `SBonus.Ошибка` — хатолар
